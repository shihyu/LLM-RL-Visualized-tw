# 歡迎建議

## 歡迎建議
- 歡迎為本專案提交糾錯與修正或其他任何改進！繪圖模板示例：[images-template.pptx](./files/images-template.pptx) 
- 提交步驟：（1）Fork： 點選頁面上的"Fork" 按鈕，在你的主頁建立獨立的子儲存庫 → （2）Clone：將你Fork後的子儲存庫Clone到本地 → （3）本地新建分支 → （4）修改提交 → （5）Push到遠端子儲存庫 → （6）提交PR：回到Github頁面，在你的子儲存庫介面點選“Compare & pull request”發起PR，等待維護者稽核&合併到母儲存庫。
